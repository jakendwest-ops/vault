# PTHub — HOME

**What it is:** A centralised dashboard for personal trainers to manage all client data in one place.  
**Who for:** Personal trainers working with multiple clients.  
**The job it does:** Replaces scattered spreadsheets/notes with a single hub — client profiles, goals, weight tracking, session notes.

## Map
- [[STATUS]] — live state and next action
- [[LOG]] — session history
- [[CRITICAL]] — hard facts and constraints

## Vision
Start as a local HTML file → evolve to a hosted website → eventually a full app. No backend yet; currently all in-memory JavaScript.
